StreamLink Ticket System 🎟️

Project Description

StreamLink Ticket System is a simple web application used to book and manage tickets online. Users can create tickets, view ticket details, and manage requests easily through a graphical interface.

Features

- Create a new ticket
- View ticket details
- Manage ticket requests
- User-friendly interface
- Simple ticket tracking system

Technologies Used

- ServiceNow Developer

How to Use

1. Open the application.
2. Login with your user credentials.
3. Create a new ticket using the form.
4. Submit the ticket request.
5. View or track the ticket status.

Project Structure

- Backend: developer
- Database: ServiceNow Tables
